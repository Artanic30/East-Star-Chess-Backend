# East-Star-Chess-Backend

## 1. Description

> Backend for East-Star-Chess platform with Django

## 2. Requirements

    Django >= 2.1.7
    djangorestframework >= 3.10.2

## 3. Build Setup

``` bash

python3 manage.py makemigrations

python3 manage.py migrate

```

## 4. Database
    sqlite3
